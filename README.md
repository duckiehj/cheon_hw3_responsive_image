# HW2 Responsive Scaffold
This repository contains the submission for MMED-1055 Assignment #3 Responsive Image Sizing by Jin Cheon. 
The project focuses on implementing responsive images within a basic grid layout using HTML and CSS. Three sets of images are created for desktop, tablet, and mobile screen sizes. Each set demonstrates a different responsive image technique, including the use of the img tag with srcset, the picture element, and responsive background images in CSS. Media queries are used to ensure that the appropriate image is displayed at each screen size.
